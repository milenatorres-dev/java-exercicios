Este repositório contém uma coleção de exercícios desenvolvidos em Java com foco no aprendizado de lógica de programação, estruturas de controle e organização de código.

O projeto foi estruturado seguindo boas práticas utilizadas no mercado, incluindo separação por pacotes e responsabilidades.

---

## 🚀 Tecnologias utilizadas

* Java (JDK 8+)
* IntelliJ IDEA
* Scanner (entrada de dados)
* Random (geração de números aleatórios)

## 📘 Descrição dos Exercícios

### 🔹 Pacote: `basicos`

#### ✔️ Conversão de Temperatura

Converte uma temperatura de Celsius para Fahrenheit.

**Conceitos:**

* Tipos primitivos
* Operações matemáticas
* String.format
* Casting

**Fórmula:**
F = (C × 1.8) + 32

---

#### ✔️ Cálculo de Fatorial

Calcula o fatorial de um número informado.

**Conceitos:**

* Estrutura de repetição (for)
* Variável acumuladora

---

#### ✔️ Fatorial com detalhamento

Mostra o cálculo completo do fatorial (ex: 5! = 5 x 4 x 3 x 2 x 1 = 120).

**Conceitos:**

* StringBuilder
* Construção dinâmica de string

---

### 🔹 Pacote: `jogos`

#### 🎯 Jogo de Adivinhação (for)

Usuário tenta acertar um número aleatório em até 5 tentativas.

#### 🎯 Jogo de Adivinhação (while)

Versão com controle de tentativas e feedback mais completo.

**Conceitos:**

* Random
* Condicionais
* Estruturas de repetição

---

### 🔹 Pacote: `menu`

#### 📐 Cálculo de Áreas

Menu interativo para calcular:

* Área do quadrado
* Área do círculo

**Conceitos:**

* Métodos
* Switch-case
* Reutilização de código

---

#### 🏦 Operações Bancárias

Simulação de um sistema bancário simples com:

* Consulta de saldo
* Depósito
* Transferência

**Conceitos:**

* Menu interativo
* Estrutura de repetição (while)
* Controle de fluxo

---
#### 🚗 Cálculo de Autonomia de Veículo
Este projeto em Java calcula a autonomia de um veículo com base no consumo médio e na quantidade de combustível disponível.

#### Funcionalidades:
- Cálculo da autonomia máxima (tanque cheio)
- Cálculo da autonomia atual
- Verificação se o combustível é suficiente para a viagem
- Cálculo da quantidade de combustível necessária para completar o trajeto
- Estimativa de litros que precisam ser abastecidos
